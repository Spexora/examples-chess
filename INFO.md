# Chess

This is an open-source, easy to set up chess application.

This application should have a nice dark/white theme.

Hoever effects on the board, hover effect on the board pieces, and movement animations.

It should be powered by Svelte Kit.

Moves get sent to the server by API, and validated by the server.

State gets sent to the clients by SSE.

There should also be a chat where the opponents can chat with each other.

Before a game starts, the "host" can choose if they want to start as black or white, then they'll see the invite link.

When the 2nd user joins, the game starts.
